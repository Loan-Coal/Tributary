## Makefile target specifications

```makefile
test:          pytest tests/ --cov=src/tributary --cov-report=term-missing -q
test-engine:   pytest tests/ -k "engine" --cov=src/tributary/engine --cov-report=term-missing -q
ingest:        python -m tributary.ingestion.cli
run-golden:    python -m tributary.engine.cli run-golden
demo:          python -m tributary.engine.cli demo
check-layers:  python scripts/check_layers.py
```

The `check_layers.py` script should grep for:
- `neo4j` in any file outside `src/tributary/graph/`
- `anthropic` in any file outside `src/tributary/ai/`
- Any import of `src/tributary/engine` in `src/tributary/ai/`
- Any import of `src/tributary/ai` in `src/tributary/engine/`
(Stub implementation is acceptable — real enforcement via CI later.)
